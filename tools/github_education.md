# GitHub Education 가이드

## GitHub Education이란?
GitHub Education은 학생들에게 개발 도구를 무료로 제공하는 프로그램입니다.<br>
개인적으로 적극 활용하는 것을 추천드립니다.

## 혜택 및 도구

### GitHub Student Developer Pack
- **GitHub Pro**: 무제한 Public/Private 저장소
- **Copilot**: AI 코드 완성 (최초 60일 무료, 이후 학생 무료)
- **Codespaces**: 클라우드 개발 환경 (월 180시간 무료)

### 데이터/환경 관련 혜택
- **AWS Credits**: $150/년
- **Google Cloud Credits**: $50/월
- **JetBrains IDE**: IntelliJ, PyCharm 등 무료 라이선스
- **Digital Ocean**: $50/월 크레딧

## 가입 방법

1. GitHub 계정 생성 (이메일은 대학 메일 권장)
2. [GitHub Education](https://education.github.com/) 방문
3. "Benefits" → "Get student benefits" 클릭
4. 학생 신분증/수강 증명서 제출 (저는 학생증으로 하니까 안돼서 영문 재학증명서 뽑아서 했습니다)

## 활용 팁

- Codespaces를 사용해 로컬 환경 설정 없이 클라우드에서 개발 가능
- AWS/GCP 크레딧으로 GPU 학습 비용 절감
- Copilot으로 생산성 향상
