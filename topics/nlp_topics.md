# NLP 추천 주제

## 선정 기준
- 실무 활용도가 높을 것
- 데이터와 베이스라인이 공개되어 있을 것
- 4주 내에 의미 있는 실험이 가능할 것

## 주의 사항
- 해당 주제는 추천 자료일 뿐, 반드시 이 중에서 선택해야 하는 것은 아닙니다.
- 자신만의 아이디어나 관심 분야를 선택해도 좋습니다.
- 제가 재밌어보이는거 생각나는대로 적은거라서 참고만 해주시고, 실제 주제 선정은 각 팀의 실력과 선호에 맞게 선정하시면 됩니다.
- 난이도는 gpt에 물어본거라서 절대적으로 믿지는 말아주세요.

## 추천 주제 목록

### [MLOps/분류] 다중 레이블 악플 및 편향 탐지 시스템
- **설명**: 단순 욕설을 넘어 혐오, 인종/지역/성별 편향 등 7~10가지 카테고리로 세분화된 악플 탐지
- **데이터셋**: [AI Hub 텍스트 윤리검증 데이터](https://aihub.or.kr/aihubdata/data/view.do?currMenu=115&topMenu=100&aihubDataSe=realm&dataSetSn=558), [korean-hate-speech](https://github.com/kocohub/korean-hate-speech)
- **참고 레포**: [ratsgo/nlpbook](https://github.com/ratsgo/nlpbook)(텍스트분류 튜토리얼), [KcBERT](https://github.com/Beomi/KcBERT), [Hugging Face Optimum](https://huggingface.co/docs/optimum/index)
- **난이도**: ★★★☆☆ (중급)
- 단순히 욕이냐 아니냐를 넘어서 혐오, 편향, 공격성 등 다양한 카테고리로 악플을 탐지하는 시스템입니다. 실제로 네이버나 다음 같은 포털 사이트에서 댓글을 관리하는 시스템에 적용할 수 있을 것 같습니다. 이를 실제 서비스에 올릴 수 있을 만큼 가볍게 만드는 경량화까지 고민해보면 더 좋을 것 같습니다.

### [임베딩/검색] 그 단어 뭐더라? 감정/느낌 설명 → 단어 추천 시스템
- **설명**: "비 오는 날의 쓸쓸한 느낌"처럼 문장을 입력하면 국어사전의 뜻풀이를 검색하여 그에 맞는 단어 추천, Sentence-BERT로 의미 기반 임베딩 검색 시스템을 구축
- **데이터셋**: 국립국어원 우리말샘 API, KorSTS(한국어 의미 유사도 데이터셋)
- **참고 레포**: [Sentence-Transformers](https://github.com/UKPLab/sentence-transformers), [FAISS](https://github.com/facebookresearch/faiss)
- **난이도**: ★★★☆☆ (중급)
- 가끔식 단어가 생각이 안날때 느낌 설명으로 단어를 추천해주는 시스템이 있으면 좋겠다는 생각이 들었습니다. 실제로 이런 시스템이 있으면 글쓰기나 대화에서 유용하게 활용할 수 있을 것 같습니다.

### [LLM/RAG] 부동산 임장 도우미 LLM
- **설명**: 학교 주변 자취 정보나 중고 거래 팁을 RAG 방식으로 구현하여, 사용자 질문에 대해 임장 조언과 팁을 제공하는 챗봇
- **데이터셋**: 부동산 사이트(직방, 다방 등), 임장 체크리스트, 자취 팁 데이터 (자체 구축)
- **참고 라이브러리**: langchain, OpenAI API, langgraph, chromadb 등 
- **난이도**: ★★★★☆ (중급~고급)
- 이번에 자취방을 구하게 되면서 실제로 이런 챗봇이 있으면 좋겠다는 생각이 들었습니다. 실용성도 높고 매력적인 포트폴리오 주제가 될 수 있을 것 같습니다. 다만, 데이터 수집이 딱봐도 어려워 보여서 학기 중에는 구현에 초점을 맞추고, 방학에 추가로 데이터 수집과 모델 개선을 해보는 것도 좋을 것 같습니다.
