## Introduction
A.ing 시니어 트랙은 실전 실무 역량 강화와 포트폴리오 확보를 목표로 합니다.<br>
이 트랙에서는 CV, NLP, RL과 같은 주제를 다루며, 프로젝트의 완성도보다는 체계적으로 기록하는 것을 목표로 진행합니다.<br>
1학기 당 1개의 프로젝트를 진행하며, 팀의 실력이나 선호도에 맞춰 2개도 가능합니다. <br>

## Eligibility
### Required
- Python 코드를 읽고 기본 데이터 처리 및 모델 학습을 위해 수정할 수 있는 능력
- 로컬 Jupyter Notebook 또는 Python 스크립트 실행 환경을 설정할 수 있는 능력
- 팀 프로젝트에 적극적으로 참여하고 지속할 수 있는 의지
- 기본 딥러닝 지식 (모델, 손실 함수, 학습 과정 등)
### Recommended
- 선택한 주제(CV, NLP, RL)에 대한 기본 이해
- 논문을 읽을 수 있는 능력
- GitHub 리포지토리를 클론하고 실행할 수 있는 기본 Git 경험
- 주니어 트랙 수료 또는 유사한 프로젝트 경험 (예: Kaggle 대회 참여, 개인 프로젝트 등)
### Expected Outcomes
- 단순히 코드를 실행하는 것을 넘어, 자신의 데이터와 환경에 맞춰 코드를 수정하고 개선하는 능력
- 향후 취업이나 연구에 활용할 수 있는 재현 가능한 포트폴리오 구축
- 머리로만 알고 있던 이론과 실제 구현 사이의 격차를 줄이는 경험
- 프로젝트를 이끌거나 팀원과 협업하는 과정에서 커뮤니케이션 능력 향상
