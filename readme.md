## Introduction
A.ing 시니어 트랙은 실전 실무 역량 강화와 포트폴리오 확보를 목표로 합니다.<br>
이 트랙에서는 CV(컴퓨터 비전), NLP(자연어 처리), RL(강화 학습)과 같은 주제를 다루며, 완성도보다는 체계적으로 기록하는 것을 평가 기준으로 삼습니다. <br>
4주간 1개의 주제로 진행되며, 1학기 동안 2개의 프로젝트를 완성하는 것을 목표로 합니다.

## Eligibility
### Required
- Python 코드를 읽고 기본 데이터 처리 및 모델 학습을 위해 수정할 수 있는 능력
- 로컬 Jupyter Notebook 또는 Python 스크립트 실행 환경을 설정할 수 있는 능력
- 팀 프로젝트에 적극적으로 참여하고 지속할 수 있는 의지

### Recommended
- 기본 딥러닝 지식 (모델, 손실 함수, 학습 과정 등)
- 선택한 주제(CV, NLP, RL)에 대한 기본 이해
- GitHub 리포지토리를 클론하고 실행할 수 있는 기본 Git 경험

## Weekly Plan
| 주차 | 내용 |
|------|------|
| 1주차 (기획 및 탐색) | 주제 선정 및 깃허브 리포지토리 찾기, 코드 실행 및 구조 파악 |
| 2주차 (튜닝) | 남의 코드에 내 데이터 넣어보기, 데이터 로더 수정 및 전처리, 에러 해결 |
| 3주차 (최적화) | 모델 구조 개선, 하이퍼파라미터 튜닝, 웹 데모 붙이기, 블로그 등 기록하기 |
| 4주차 (발표) | 최종 결과물 발표, 피드백 및 개선점 논의 |
