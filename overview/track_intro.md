# A.ing 시니어 트랙 소개

A.ing 시니어 트랙은 실전 실무 역량 강화와 포트폴리오 확보를 목표로 합니다. CV(컴퓨터 비전), NLP(자연어 처리), RL(강화 학습) 중 하나의 주제를 선택해 4주간 프로젝트를 진행합니다. 완성도보다 과정 기록의 체계성과 재현 가능성을 더 중요하게 평가합니다.

## 트랙 목표
- 실전 프로젝트를 통해 문제 정의부터 결과물 정리까지의 흐름을 경험
- 실험과 의사결정 과정을 문서로 정리해 재현 가능한 포트폴리오 제작
- 오픈소스 레포지토리를 분석하고 내 데이터/환경에 맞게 개선

## 진행 방식
- 주차별 목표에 맞춰 작업하고, 기록과 산출물을 제출
- 팀 단위 운영 원칙, 인원 부족 시 개인 참여 가능
- 제공 자료: 추천 GitHub 레포 3개, README 템플릿, 추천 주제, 도구 정보
