# A.ing 시니어 트랙 소개
A.ing 시니어 트랙은 실전 실무 역량 강화와 포트폴리오 확보를 목표로 합니다. CV, NLP, RL 중 하나의 주제를 선택해 4주간 프로젝트를 진행합니다. 완성도보다 과정 기록의 체계성과 재현 가능성을 더 중요하게 두고 진행합니다.

## 트랙 목표
- 실전 프로젝트를 통해 문제 정의부터 결과물 정리까지의 흐름을 경험
- 실험과 의사결정 과정을 문서로 정리해 재현 가능한 포트폴리오 제작
- 오픈소스 레포지토리를 분석하고 내 데이터/환경에 맞게 개선

## 진행 방식
- 주차별 목표에 맞춰 작업하고, 기록과 산출물 제출
- 팀 단위 운영 원칙, 인원 부족 혹은 관심 분야가 맞지 않는 경우 개인 참여 가능
- 별도의 절대 평가는 없으며, 학기 말 발표와 투표를 통해 가장 많은 투표를 받은 팀에게 회식비 10만원 지원
- 제공 자료: 추천 GitHub 레포 3개, README 템플릿, 추천 주제, 도구 정보
